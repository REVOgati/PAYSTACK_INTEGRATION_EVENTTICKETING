✓ ENDPOINTS IMPLEMENTED:

Auth (POST /auth):
  - POST /auth/register → Create user account
  - POST /auth/login → Get JWT access token

Events (GET/POST/PATCH /events):
  - GET /events → List published events
  - GET /events/{event_id} → Get event details
  - POST /events → Create event (organizer only)
  - PATCH /events/{event_id} → Update event (organizer only)

Bookings (POST/GET /bookings):
  - POST /bookings → Reserve tickets (creates pending booking)
  - GET /bookings/{booking_id} → Get booking details
  - GET /bookings/me → Get user's all bookings
  - GET /bookings/{booking_id}/tickets → Retrieve issued tickets

Payments (POST/GET /payments):
  - POST /payments/initialize → Call Paystack, get authorization URL
  - GET /payments/callback → Handle Paystack redirect
  - GET /payments/verify/{reference} → Manual payment verification

Webhooks (POST /webhooks):
  - POST /webhooks/paystack → Receive Paystack charge.success/failed events
    * Validates HMAC signature
    * Confirms bookings on success
    * Issues tickets automatically
    * Releases tickets on failure

Health:
  - GET /health → API health status
